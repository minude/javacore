      # jconsole

      jconsole Тў» JDK УЄфтИдуџё GUI тиЦтЁисђѓjconsole(Java Monitoring and Management Console) Тў»СИђуДЇтЪ║С║ј JMX уџётЈ»УДєтїќуЏЉУДєСИју«АуљєтиЦтЁисђѓ

      jconsole уџёу«АуљєтіЪУЃйТў»жњѕт»╣ JMX MBean У┐ЏУАїу«Ауљє№╝їућ▒С║ј MBean тЈ»С╗ЦСй┐ућеС╗БуаЂсђЂСИГжЌ┤С╗ХТюЇтіАтЎеуџёу«АуљєТјДтѕХтЈ░ТѕќТЅђТюЅугдтљѕ JMX УДёУїЃуџёУй»С╗ХУ┐ЏУАїУ«┐жЌ«сђѓ

      Т│еТёЈ№╝џСй┐уће jconsole уџётЅЇТЈљТў» Java т║ћућет╝ђтљ» JMXсђѓ

      # т╝ђтљ» JMX

      т╝ђтљ»Тќ╣Т│ЋТў»№╝їтюе java ТїЄС╗цтљј№╝їТи╗тіаС╗ЦСИІтЈѓТЋ░№╝џ

      # MAT

      MAT тЇ│ Eclipse Memory Analyzer Tool уџёу╝ЕтєЎсђѓ

      MAT ТюгУ║ФС╣ЪУЃйтцЪУјитЈќтаєуџёС║їУ┐ЏтѕХт┐ФуЁДсђѓУ»ЦтіЪУЃйт░єтђЪтіЕ jps тѕЌтЄ║тйЊтЅЇТГБтюеУ┐љУАїуџё Java У┐ЏуеІ№╝їС╗ЦСЙЏжђЅТІЕт╣ХУјитЈќт┐ФуЁДсђѓућ▒С║ј jps С╝џт░єУЄфти▒тѕЌтЁЦтЁХСИГ№╝їтЏаТГцСйаС╝џтюетѕЌУАеСИГтЈЉуј░СИђСИфти▓у╗Ју╗ЊТЮЪУ┐љУАїуџё jps У┐ЏуеІсђѓ

      MAT тЈ»С╗ЦуІгуФІт«ЅУБЁ№╝ѕт«ўТќ╣СИІУййтю░тЮђ№╝Ѕ№╝їС╣ЪтЈ»С╗ЦСйюСИ║ Eclipse IDE уџёТЈњС╗Хт«ЅУБЁсђѓ

      # MAT жЁЇуй«

      MAT УДБтјІтљј№╝їт«ЅУБЁуЏ«тйЋСИІТюЅСИф MemoryAnalyzer.ini ТќЄС╗Хсђѓ

      MemoryAnalyzer.ini СИГТюЅСИфжЄЇУдЂуџётЈѓТЋ░ Xmx УАеуц║ТюђтцДтєЁтГў№╝їж╗ўУ«цСИ║№╝џ-vmargs -Xmx1024m

      тдѓТъюУ»ЋтЏЙуће MAT т»╝тЁЦуџё dump ТќЄС╗ХУХЁУ┐Є 1024 M№╝їС╝џТіЦжћЎ№╝џ

      # MAT тѕєТъљ

      уѓ╣тЄ╗ Leak Suspects тЈ»С╗ЦУ┐ЏтЁЦтєЁтГўТ│ёТ╝ЈжАхжЮбсђѓ

      №╝ѕ1№╝ЅждќтЁѕ№╝їтЈ»С╗ЦТЪЦуюІжЦ╝тЏЙС║єУДБтєЁтГўуџёТЋ┤СйЊТХѕУђЌТЃЁтєх

      №╝ѕ2№╝Ѕу╝Ет░ЈУїЃтЏ┤№╝їт»╗ТЅЙжЌ«жбўуќЉС╝╝уѓ╣

      тЈ»С╗Цуѓ╣тЄ╗У┐ЏтЁЦУ»дТЃЁжАхжЮб№╝їтюеУ»дТЃЁжАхжЮб Shortest Paths To the Accumulation Point УАеуц║ GC root тѕ░тєЁтГўТХѕУђЌУЂџжЏєуѓ╣уџёТюђуЪГУи»тЙё№╝їтдѓТъюТЪљСИфтєЁтГўТХѕУђЌУЂџжЏєуѓ╣ТюЅУи»тЙётѕ░УЙЙ GC root№╝їтѕЎУ»ЦтєЁтГўТХѕУђЌУЂџжЏєуѓ╣СИЇС╝џУбФтйЊтЂџтъЃтюЙУбФтЏъТћХсђѓ

      СИ║С║єТЅЙтѕ░тєЁтГўТ│ёжю▓№╝їТѕЉУјитЈќС║єСИцСИфтаєУйгтѓеТќЄС╗Х№╝їСИцСИфТќЄС╗ХУјитЈќТЌХжЌ┤жЌ┤жџћТў»СИђтцЕ№╝ѕтЏаСИ║тєЁтГўтЈфТў»т░Јт╣Ёт║дтбъжЋ┐№╝їуЪГТЌХжЌ┤тЙѕжџЙтЈЉуј░жЌ«жбў№╝Ѕсђѓт»╣Т»ћСИцСИфТќЄС╗Хуџёт»╣У▒А№╝їжђџУ┐Єт»╣Т»ћтљјуџёу╗ЊТъютЈ»С╗ЦтЙѕТќ╣СЙ┐т«џСйЇтєЁтГўТ│ёжю▓сђѓ

      MAT тљїТЌХТЅЊт╝ђСИцСИфтаєУйгтѓеТќЄС╗Х№╝їтѕєтѕФТЅЊт╝ђ Histogram№╝їтдѓСИІтЏЙсђѓтюеСИІтЏЙСИГТќ╣ТАє 1 ТїЅжњ«ућеС║јт»╣Т»ћСИцСИф Histogram№╝їт»╣Т»ћтљјтюеТќ╣ТАє 2 тцёжђЅТІЕ Group By package№╝їуёХтљјт»╣Т»ћтљёт»╣У▒АуџётЈўтїќсђѓСИЇжџЙтЈЉуј░ heap3.hprof Т»ћ heap6.hprof т░ЉС║є 64 СИф eventInfo т»╣У▒А№╝їтдѓТъют»╣С╗БуаЂТ»ћУЙЃуєЪТѓЅуџёУ»ЮТЃ│т┐ЁУ┐ЎТаиСИђСИфу╗ЊТъюТў»УЃйтцЪу╗ЎуеІт║ЈтЉўСИђт«џуџётљ»уц║уџёсђѓУђїТѕЉС╣ЪТў»Та╣ТЇ«У┐ЎСИфтљ»уц║ти«ТЅЙтѕ░С║єТюђу╗ѕтєЁтГўТ│ёжю▓уџёСйЇуй«сђѓ
